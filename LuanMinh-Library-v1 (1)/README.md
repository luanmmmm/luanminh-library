# Luận Minh Library - Educational Document Repository

Dự án thư viện số cá nhân chuyên nghiệp và tối ưu hóa để lưu trữ, quản lý giáo án điện tử, tài liệu PowerPoint và PDF phục vụ công tác giảng dạy tổ môn Lịch sử & Địa lý.

## Cấu trúc thư mục mã nguồn
- `index.html`: Cấu trúc khung trang và bố cục giao diện.
- `style.css`: Hệ thống thiết kế UI/UX hiện đại hỗ trợ tự động giao diện sáng/tối (Dark/Light mode).
- `script.js`: Công cụ tìm kiếm, lọc danh mục động cùng logic điều phối modal tương tác.

## Hướng dẫn đưa lên GitHub và Vercel

1. **Chuẩn bị file:** Giải nén tệp tin `LuanMinh-Library-v1.zip` vừa tải về.
2. **Đưa lên GitHub:**
   - Truy cập kho lưu trữ: `https://github.com/luanmmmm/luanminh-library`
   - Chọn **Add file** -> **Upload files**.
   - Kéo trực tiếp các tệp tin (`index.html`, `style.css`, `script.js`, `README.md` cùng các thư mục trống nếu có) vào khu vực tải lên. *Lưu ý không tải trực tiếp cả thư mục tổng hay file zip*.
   - Nhấn **Commit changes** ở cuối trang.
3. **Triển khai Vercel:**
   - Truy cập dashboard Vercel của bạn, kết nối lại hoặc tạo dự án mới liên kết trực tiếp tới repository `luanminh-library`.
   - Chọn thiết lập mặc định và nhấn **Deploy**. Hệ thống tự động triển khai trong vòng 1 phút.
